=======================================================================================
	       			  Aplikasi I-Library
=======================================================================================
Aplikasi I-Library buku ini merupakan program 
komputer yang dirancang khusus untuk mengelola 
data – data peminjaman dan pengembalian buku 
agar dapat disajikan dengan lebih cepat

Cara Penggunaa :
- Klik Icon I-Library 
  (akan muncul tampilan login)
- Masukan username dan password 
  (jika anda tidak memiliki akun klik signup)
- Username dan password yang dimasukan akan menentukan apakah anda anggota atau petugas
  (layar utama anggota dan petugas memiliki fungsionalitas yang berbeda)
- Di layar utama terdapat banyak tab diatasnya 
  (setiap tab memilki fungsi yang berbeda)
